# Nanosatellite Telemetry Project

This project simulates a nanosatellite telemetry system using Raspberry Pi.

## Hardware
- Raspberry Pi
- BMP280 (temperature and pressure sensor)
- DHT 11 Sensor
- GPS module
- Jumper wires and breadboard

## Communication
Sensor data will be transmitted using WiFi.

## Objectives
- Read environmental data from sensors
- Process the data using Python
- Transmit telemetry data
- Maintain version control using GitHub
